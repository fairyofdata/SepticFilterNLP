# 정화조 (SepticFilter) — 딥러닝 기반 우회 욕설 감지 필터 🤬🤖

**🌐 Available Versions:** [🇺🇸 English](/README.md) | [🇯🇵 日本語 (Japanese)](/README_JP.md)

> 🏆 **삼성SDS 멀티캠퍼스 빅데이터 활용 공모전 — 최우수상 (1위)** (멀티캠퍼스, 2022)  
> K-디지털 트레이닝 데이터 사이언스 과정 팀 프로젝트

> 📌 **후속 프로젝트:** 이 프로젝트에서 쌓은 한국어 도메인 특화 NLP 경험(자모 토큰화 전략, 앙상블 설계, 커뮤니티 크롤링)은 이후 [AntiPhishingNLP](https://github.com/fairyofdata/AntiPhishingNLP)(KISA 최우수상)의 스미싱 탐지 모델 설계에 직접 적용됐습니다.

고정된 금칙어 리스트 방식의 한계를 넘어, 자모 분리·특수문자 치환 등 **변형 욕설(우회 표현)**까지 감지하는 딥러닝 필터 시스템입니다.

---

## 🎯 핵심 문제 정의

기존 필터링 시스템(금칙어 리스트)은 `씨발 → ㅅㅂ`, `개새끼 → 개ㅅㄲ`, `미1친` 같은 **자모 분리·변형 표현**을 탐지하지 못합니다. 또한 형태소 단위 분석기 역시 변형된 자모 조합을 미등록 토큰(OOV)으로 처리하여 정보를 잃어버립니다.

이 프로젝트는 다음 질문에서 출발합니다:
> *"형태소 분석기가 인식하지 못하는 변형 욕설은 어떻게 탐지할 것인가?"*

---

## 🏗️ 핵심 설계: 이중 토큰화 전략 및 소프트보팅 앙상블

형태소 분석기는 `ㅅㅂ`을 인식하지 못하지만, 자모 단위로 분해(`['ㅅ', 'ㅣ', 'ᴕ', 'ㅂ', 'ㅏ', 'ㄹ']`)하면 시퀀스가 보존되어 욕설 패턴을 학습할 수 있습니다. 

### 1차 필터링: FastText + LSTM (자모 단위)
- **FastText**를 이용해 희귀 단어 및 오타 임베딩 처리.
- 1차적으로 욕설 여부를 판별하며, 여기서 욕설로 분류되지 않은 경우 2차 필터링으로 넘어감.

### 2차 필터링: 앙상블 소프트보팅 (Soft Voting)
1. **자모 단위 LSTM (정확도 93.22%)**: 장거리 의존성 및 자모 시퀀스의 맥락 학습
2. **자모 단위 CNN (정확도 93.7%)**: `시1발` 처럼 욕설 사이에 다른 문자가 삽입된 지역적(Local) N-gram 패턴 포착

*(참고: 형태소 단위 LSTM도 구축했으나, `시@발`과 같은 텍스트를 `시`, `@`, `발`로 쪼개어 정상 단어를 욕설로 오판할 우려가 있어 최종 앙상블 보팅에서 제외함)*

| 네이트판 | 15,000개 |

### 🚨 트러블슈팅 (Crawling & Preprocessing)
- **IP Ban 차단 회피**: 크롤링 봇으로 인식되어 IP 밴을 당하는 문제 발생. `time.sleep` 값을 단순히 늘리는 대신, 최소 60초와 여러 난수의 곱을 결합한 동적 대기 시간 적용 및 `User-Agent` 헤더 스푸핑으로 해결.
- **인코딩 문제**: CSV 저장 시 한글 깨짐 현상을 `utf-8-sig` 인코딩으로 해결.
- **데이터 정제**: Python `emoji` 라이브러리로 임베딩에 악영향을 주는 이모지 제거, 정규표현식으로 댓글 수 및 불필요한 공백/줄바꿈(`\n`) 제거 후 중복 데이터 병합.

## 🖥️ 서비스 아키텍처 (Django)

- **Backend**: Django MVT 패턴을 활용한 웹 서버 구축 (`urls.py`, `views.py` 라우팅)
- **Model Serving**: `.h5` LSTM/CNN 모델 및 `.pkl` Tokenizer를 `views.py` 메모리에 로드하여 실시간 추론 (Inference)
- **DB 연동**: 사용자가 입력한 텍스트와 모델의 판별 결과(1=욕설, 0=정상)를 MySQL 데이터베이스에 자동으로 적재하여 추후 재학습 데이터로 활용 가능.
- **Frontend**: 입력 창과 실시간 판별 결과를 직관적으로 보여주는 HTML/JS 웹 UI 구현

## 🔗 후속 프로젝트와의 연결

이 프로젝트는 다음 해 **[AntiPhishingNLP](https://github.com/fairyofdata/AntiPhishingNLP)**(KISA 2023 사이버보안 챌린지 최우수상)의 직접적인 기술 기반이 됐습니다.

| 정화조 (2022) | → | AntiPhishingNLP (2023) |
|---|---|---|
| 자모 토큰화로 변형 욕설 탐지 | | 자모/형태소 이중 전략을 Att-BiLSTM에 적용 |
| CNN + LSTM + FastText 소프트보팅 | | KoBERT + KoELECTRA + Att-BiLSTM 소프트보팅 |
| 커뮤니티 댓글 크롤링 파이프라인 | | KorCCViD 정제 + SMOTE 불균형 처리 |
| Django 기반 웹 서비스 | | Django STT 실시간 보이스피싱 탐지 서버 |

앙상블 전략의 설계 철학과 한국어 특화 토큰화 경험이 그대로 계승됐습니다.
