# 정화조 (SepticFilter) — 딥러닝 기반 우회 욕설 감지 필터 🤬🤖

> 🏆 **삼성SDS 멀티캠퍼스 빅데이터 활용 공모전 — 최우수상 (1위)** (멀티캠퍼스, 2022)  
> K-디지털 트레이닝 데이터 사이언스 과정 팀 프로젝트

> 📌 **후속 프로젝트:** 이 프로젝트에서 쌓은 한국어 도메인 특화 NLP 경험(자모 토큰화 전략, 앙상블 설계, 커뮤니티 크롤링)은 이후 [AntiPhishingNLP](https://github.com/fairyofdata/AntiPhishingNLP)(KISA 최우수상)의 스미싱 탐지 모델 설계에 직접 적용됐습니다.

고정된 금칙어 리스트 방식의 한계를 넘어, 자모 분리·특수문자 치환 등 **변형 욕설(우회 표현)**까지 감지하는 딥러닝 필터 시스템입니다.

---

## 🎯 핵심 문제 정의

기존 필터링 시스템은 `씨발 → ㅅㅂ`, `개새끼 → 개ㅅㄲ`, `병신 → 뵹신` 같은 **자모 분리·변형 표현**을 탐지하지 못합니다. 이 현상이 발생하는 이유는 형태소 단위 분석기가 변형된 자모 조합을 애초에 알 수 없는 토큰으로 처리하기 때문입니다.

이 프로젝트는 다음 질문에서 출발합니다:

> *"형태소 분석기가 인식하지 못하는 변형 욕설은 어떻게 탐지할 것인가?"*

---

## 🏗️ 핵심 설계: 이중 토큰화 전략

```
[입력 문장]
    ├── 형태소 단위 토큰화 (MeCab/Komoran)
    │       → 문맥 이해에 강함, 정상 텍스트 분류 담당
    └── 자모 단위 토큰화 (Character-level Jamo decomposition)
            → ㅅㅂ / ㅂㅅ / ㄱㅅㄲ 등 변형 욕설 패턴 직접 포착
                    ↓
            [LSTM + CNN + FastText 소프트보팅 앙상블]
                    ↓
              욕설 / 정상 분류 (정확도 93.7%)
```

**왜 자모 단위 토큰화가 필요했는가:**  
형태소 분석기는 `ㅅㅂ`을 미등록 토큰으로 처리하여 정보를 버립니다. 반면 자모 단위로 분해하면 `ㅅ-ㅂ` 시퀀스가 그대로 유지되어 욕설 패턴을 학습할 수 있습니다. 이 이중 전략이 본 프로젝트의 핵심 기여입니다.

## 📊 데이터 파이프라인

### 수집 출처 (총 40,000개 댓글)
| 커뮤니티 | 수집 대상 |
|---|---|
| 디씨인사이드 (야구갤, 해연갤) | 게임·스포츠 욕설 패턴 |
| 에펨코리아 (롤, 메이플 게시판) | 게임 커뮤니티 욕설 |
| 네이트판 | 일반 커뮤니티 욕설 |
| igloos | 구형 블로그 커뮤니티 |

- **라벨링**: 수집 댓글을 욕설(1) / 정상(0)으로 수동 이진 라벨링
- **변형 보존 전처리**: 특수문자·이모지·숫자를 제거하지 **않고** 그대로 학습에 활용 (변형 욕설 탐지가 목적이므로)

## 🔬 모델 구성

### 앙상블 구성원

| 모델 | 토크나이저 | 역할 |
|---|---|---|
| **LSTM** | 자모 단위 | 자모 시퀀스에서 장거리 패턴 학습 |
| **CNN** | 자모 단위 | N-gram 수준의 지역 특징(변형 욕설 패턴) 포착 |
| **FastText + LSTM** | 형태소 단위 | 서브워드 임베딩으로 OOV 처리 + 문맥 이해 |

소프트보팅(Soft Voting)으로 세 모델의 확률값을 평균하여 최종 판별. 단일 모델 대비 변형 욕설 recall이 유의미하게 향상됐습니다.

### 성능
- **정확도**: 93.7%
- 특수문자·자모 분리 변형 욕설에서도 안정적 탐지 확인

## 🔗 후속 프로젝트와의 연결

이 프로젝트는 다음 해 **[AntiPhishingNLP](https://github.com/fairyofdata/AntiPhishingNLP)**(KISA 2023 사이버보안 챌린지 최우수상)의 직접적인 기술 기반이 됐습니다.

| 정화조 (2022) | → | AntiPhishingNLP (2023) |
|---|---|---|
| 자모 토큰화로 변형 욕설 탐지 | | 자모/형태소 이중 전략을 Att-BiLSTM에 적용 |
| CNN + LSTM + FastText 소프트보팅 | | KoBERT + KoELECTRA + Att-BiLSTM 소프트보팅 |
| 커뮤니티 댓글 크롤링 파이프라인 | | KorCCViD 정제 + SMOTE 불균형 처리 |
| Django 기반 웹 서비스 | | Django STT 실시간 보이스피싱 탐지 서버 |

앙상블 전략의 설계 철학과 한국어 특화 토큰화 경험이 그대로 계승됐습니다.
